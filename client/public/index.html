About SmartFarm

Born from the Farm. Built for the Future.

By combining IoT hardware (ESP32 microcontrollers, DHT11 sensors, soil moisture probes) with a cloud-connected web platform, SmartFarm gives every farmer — from a small quarter-acre plot to a large commercial operation — the power to monitor their fields and control irrigation from a smartphone or computer, anytime, anywhere.

Our mission is simple: reduce water waste, increase crop yields, and put the tools of precision agriculture within reach of every farmer.

How It Works

From connecting sensors to harvesting bigger yields in four simple steps.

1

📡 Connect Your Sensors

Install the ESP32 microcontroller with soil moisture, temperature, and humidity sensors in your field. Or enter readings manually through the web dashboard.

2

📊 Monitor in Real-Time

Your farm dashboard updates live with every new reading. View trends, charts, and current conditions from any device, anywhere in the world.

3

🔔 Receive Smart Alerts

The system automatically checks your readings against your configured thresholds and sends instant alerts when conditions need your attention.

4

💧 Automate Irrigation

When soil moisture drops below your threshold, SmartFarm automatically triggers your water pump. Schedule watering windows or control the pump manually.

Under the Hood

Modern Stack for Precision Farming

Hardware Layer
  • ESP32 Microcontroller (WiFi + Bluetooth)
  • DHT11 Temperature & Humidity Sensor
  • Capacitive Soil Moisture Sensor
  • 5V Relay Module for pump control
  • 12V Submersible Water Pump
Software Layer
  • Node.js + Express.js backend API
  • MongoDB Atlas cloud database
  • Socket.IO for real-time data streaming
  • JWT authentication with role-based access
  • Automated cron-based irrigation scheduling
Dashboard Layer
  • Real-time sensor cards with live updates
  • Interactive historical charts (Chart.js)
  • Multi-portal system (Admin / Farmer / Viewer)
  • Mobile-responsive web interface
  • No app installation required
